## üöÄ **Getting Started**

### **Installation**
```bash
# Clone the repository
git clone https://github.com/EverettNC/yorkiepatty.git
cd yorkiepatty

# Install dependencies
pip install -r requirements.txt

# Configure AWS credentials for Polly
aws configure

# Run the system
python main.py
```

### **Quick Start**
```bash
# Launch Sonny's teaching interface
python main.py --mode=teaching

# Start web interface
python app.py

# Run Yorkie's progress dashboard  
python learning_progress_tracker.py --student=yorkie
```
